# ابطال الرياضيات - Math Heroes 🎮

منصة تعليمية تفاعلية باللغة العربية لتعلم الرياضيات بطريقة ممتعة!

## 🌟 المميزات

- **واجهة عربية كاملة (RTL)** مع خط Cairo الجميل
- **لعبة آلة الضعف والنصف** التفاعلية
- **مؤثرات بصرية وصوتية** (Confetti, Animations, Custom Sounds)
- **لوحة تحكم للمعلم** لمتابعة نتائج الطلاب
- **إدارة الأصوات** من لوحة التحكم
- **تصميم عصري وجذاب** مع ألوان زاهية

## 🛠️ التقنيات المستخدمة

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **MongoDB** with Mongoose
- **NextAuth.js** للمصادقة
- **Canvas Confetti** للمؤثرات البصرية

## 📋 المتطلبات

- Node.js 18+
- MongoDB (محلي أو سحابي)

## 🚀 التثبيت والتشغيل

### 1. تثبيت المكتبات

```bash
npm install
```

### 2. إعداد قاعدة البيانات

تأكد من تشغيل MongoDB محلياً على المنفذ الافتراضي 27017، أو قم بتحديث ملف `.env.local` بعنوان قاعدة البيانات الخاصة بك.
أضف المتغيرات التالية إلى ملف `.env.local`:

```
NEXTAUTH_URL=http://localhost:3080
NEXTAUTH_SECRET=your-secret-key-change-this-in-production
```

### 3. إنشاء حساب المعلم

بعد تشغيل المشروع، قم بزيارة:

```
http://localhost:3080/api/seed
```

هذا سينشئ حساب المعلم الافتراضي:
- **البريد الإلكتروني:** admin@school.com
- **كلمة المرور:** password123

### 4. تشغيل المشروع

```bash
npm run dev
```

افتح المتصفح على: `http://localhost:3080`

## 📱 الصفحات

### 🏠 الصفحة الرئيسية (`/`)
- عرض مميزات المنصة
- أزرار للبدء باللعبة أو الدخول للوحة المعلم

### 🎮 صفحة اللعبة (`/game`)
- إدخال اسم الطالب
- 10 أسئلة عشوائية عن الضعف والنصف
- مؤثرات بصرية وصوتية
- حساب النقاط والوقت

### 📊 صفحة النتائج (`/results`)
- عرض النتيجة النهائية
- النسبة المئوية
- الوقت المستغرق
- رسائل تشجيعية

### 👨‍🏫 تسجيل دخول المعلم (`/login`)
- تسجيل دخول آمن
- عرض بيانات تجريبية

### 🎛️ لوحة التحكم (`/admin`)
- عرض نتائج جميع الطلاب
- إدارة الأصوات (صوت النجاح، الخطأ، الموسيقى الخلفية)
- تسجيل الخروج

## 🎨 التصميم

- **الاتجاه:** RTL (من اليمين لليسار)
- **الخط:** Cairo (عربي، واضح، مناسب للأطفال)
- **الألوان:** تدرجات زاهية ومبهجة
- **الأنيميشن:** Shake, Bounce, Confetti

## 🔧 التخصيص

### تغيير الأصوات

من لوحة التحكم (`/admin`)، يمكن للمعلم إدخال روابط الأصوات:
- صوت الإجابة الصحيحة
- صوت الإجابة الخاطئة
- موسيقى الخلفية

### تغيير عدد الأسئلة

في ملف `app/game/page.tsx`، عدّل المتغير:

```typescript
const totalQuestions = 10; // غيّر هذا الرقم
```
